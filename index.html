d3.chart.pie

A pie chart built using the d3.chart library

Usage Instructions

You can install d3.chart.pie view bower by running:

$ bower install

After installing you can run the following commands to fetch all dependencies:

$ npm install
$ bower install

Open the index.html file in your browser.
